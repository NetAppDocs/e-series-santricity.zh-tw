= 管理工作階段逾時
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以設定Unified Manager的逾時、讓使用者在指定時間後中斷非作用中工作階段的連線。

.關於這項工作
Unified Manager的工作階段逾時預設為30分鐘。您可以調整時間、也可以一併停用工作階段逾時。

.步驟
. 從功能表列中、選取使用者登入名稱旁的下拉式箭頭。
. 選擇*啟用/停用工作階段逾時*。
+
「啟用/停用工作階段逾時」對話方塊隨即開啟。

. 使用微調控制項來增加或減少時間（以分鐘為單位）。
+
您可以設定的最短逾時時間為15分鐘。

+
[NOTE]
====
若要停用工作階段逾時、請清除*設定時間長度...*核取方塊。

====
. 按一下「 * 儲存 * 」。
