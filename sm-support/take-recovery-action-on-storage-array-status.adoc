= 針對儲存陣列狀態採取恢復行動
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以檢閱不可讀取的磁區記錄、重新啟用磁碟機連接埠、以及清除還原模式、藉此對目前儲存陣列的狀態採取行動。
