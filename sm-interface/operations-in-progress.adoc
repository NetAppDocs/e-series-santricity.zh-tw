= 「營運進行中」頁面
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
「作業進行中」頁面可讓您檢視目前正在執行或處於暫停、停止或失敗狀態的長期執行作業清單。您可以針對特定類型的作業採取行動。

長時間執行的作業可能會影響系統效能。對於某些作業、您可以選擇停止作業或變更其優先順序、以便更重要的作業先完成。
