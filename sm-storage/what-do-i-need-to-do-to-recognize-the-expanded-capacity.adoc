= 我需要如何辨識擴充的容量？
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您增加磁碟區的容量、主機可能無法立即辨識磁碟區容量的增加。

大多數作業系統都會辨識擴充的Volume容量、並在啟動Volume擴充之後自動擴充。不過、有些可能沒有。如果作業系統無法自動辨識擴充的Volume容量、您可能需要執行磁碟重新掃描或重新開機。

擴充磁碟區容量之後、您必須手動增加檔案系統大小以符合需求。您的做法取決於所使用的檔案系統。

如需其他詳細資料、請參閱主機作業系統文件。
