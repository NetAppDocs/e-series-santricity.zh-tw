= 手動建立主機
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
建立主機是讓儲存陣列瞭解其連接的主機、以及允許I/O存取磁碟區所需的步驟之一。您只能手動建立主機。



== 手動建立

手動建立主機可確保儲存陣列控制器偵測到的主機連接埠識別碼與主機正確關聯。

在手動建立主機期間、您可以從清單中選取主機連接埠識別碼、或手動輸入識別碼、藉此建立關聯。建立主機之後、如果您打算共用對磁碟區的存取權、可以將磁碟區指派給主機叢集或將其新增至主機叢集。
