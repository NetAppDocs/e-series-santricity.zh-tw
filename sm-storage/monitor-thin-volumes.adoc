= 在 SANtricity System Manager 中監控精簡卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以監控精簡磁碟區的空間、並產生適當的警示、以防止容量不足的情況發生。

[NOTE]
====
EF600/EF600C 或 EF300/EF300C 儲存系統無法使用此功能。

====
[NOTE]
====
不提供建立精簡磁碟區的選項。SANtricity如果要建立精簡磁碟區、請使用命令列介面（CLI）。

====
