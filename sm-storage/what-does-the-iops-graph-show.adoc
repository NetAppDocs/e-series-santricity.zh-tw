= IOPS圖表顯示什麼？
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
IOPS圖表會顯示每秒輸入/輸出作業的統計資料。在首頁上、此圖表會顯示儲存陣列的統計資料。在「效能」方塊的「邏輯檢視」、「實體檢視」及「應用程式與工作負載檢視」索引標籤中、此圖表會顯示儲存陣列、磁碟區、磁碟區群組、集區、應用程式、 和工作負載。

IOPS是每秒_輸入/輸出（I/O）作業的縮寫。將游標停留在圖表上的某個點上、即可檢視該時間點的下列值：

* 讀取作業數。
* 寫入作業數。
* 合併的讀寫作業總計。
