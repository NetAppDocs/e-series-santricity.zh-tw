= 設定管理連接埠之後該怎麼辦？
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果變更儲存陣列的IP位址、您可能會想要在Unified Manager中更新全域陣列檢視。

若要在Unified Manager中更新全域陣列檢視、請開啟介面、然後前往功能表：「Manage（管理）」[Discover（探索）。

如果您仍在使用SANtricity 此功能、請前往「企業管理」視窗（EMW）、您必須在其中移除並重新新增新的IP位址。
