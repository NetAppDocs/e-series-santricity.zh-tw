= 為什麼我看不到所有的儲存陣列？
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在匯入設定作業期間、目標選擇對話方塊中可能無法使用部分儲存陣列。

儲存陣列可能不會出現、原因如下：

* 韌體版本低於8.50。
* 儲存陣列已離線。
* 系統無法與該陣列通訊（例如、陣列有憑證、密碼或網路問題）。
